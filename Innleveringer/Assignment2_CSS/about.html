Questions

  1. The server in a client-server architecture is responsible for storing and managing the data and resources that clients request. It processes client requests and returns the necessary data or services.
  2. The client in this architecture is a program or device that requests services or resources from the server. It is usually a computer, smartphone, or another device where users interact with applications that communicate with the server.
  3. An IP (Internet Protocol) address is a numerical label assigned to each device connected to a computer network. It enables the identification of devices and allows for data routing between them.
  4. DNS (Domain Name System) is a system that translates human-readable domain names, like www.vg.no, into IP addresses, which are used by computers to identify each other on the network.
  5. The Internet is a global network of interconnected computers and servers that communicate with each other using IP protocol. The World Wide Web (WWW) is a service that operates over the Internet, allowing users to access and share information through websites and webpages using browsers.
  6. HTML (HyperText Markup Language) is the standard language used to create webpages. It structures the content on the page using a series of elements represented by tags.
  7. We include CSS files inside the head element to ensure that the styles are loaded before the body content. This prevents a FOUC.
  8. Inline style CSS overrides rules defined in style elements and external stylesheets because it has the highest specificity. It is applied directly to the element, making it the most prioritized styling source.
